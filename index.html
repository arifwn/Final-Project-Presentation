Air Quality Modeling System

AQM System

AQM Web Interface

  • Antar muka grafis berbasis web
  • Koordinasi sentral semua server
  • Menghubungkan pengguna dengan model

AQM RPC Server

  • RPC: Remote Procedure Call
  • Node di dalam cluster
  • Berfungsi untuk menjalankan model
  • Menentukan kapasitas komputasi sistem

AQM System

  • Bertindak sebagai Black Box

Diagram Proses AQM System

Model dalam AQM System

WRF

Weather Research and Forecasting

  • Mesoscale meteorological model
  • Full CFD (Computational Fluids Dynamics) model.
  • Format data terstandarisasi (NetCDF).
  • Digunakan dalam berbagai aplikasi cuaca.

WRF-Chem

  • Modul tambahan yang dipasang pada WRF.
  • Memungkinkan simulasi dispersi pencemar oleh WRF.
  • Hanya efektif untuk skala meso.

AERMOD

  • US EPA Regulatory Model
  • Model Gaussian dengan Complex Terrain.
  • AERMAP tidak digunakan dalam AQM System.
